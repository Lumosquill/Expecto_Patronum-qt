# Expecto Patronum 🪄

> *呼神护卫 —— 哈利·波特主题 2D 弹幕射击游戏*

基于 **Qt 6 (C++)** 与 **QGraphicsScene / QGraphicsView** 框架构建的横版射击游戏。玩家操控哈利·波特，在霍格沃茨的禁林中释放守护神咒，抵御摄魂怪与蜘蛛的层层进攻，在倒计时中存活并走向胜利。

---

## 核心系统

### 战斗机制

| 机制 | 说明 |
|------|------|
| **魔力攻击** | 子弹伤害 = `Mana × (1 + ManaRatio)`，魔力越高输出越强 |
| **攻击速度** | 随经验成长，攻击间隔对数递减，下限 200ms |
| **防御减伤** | 受伤量 = `Attack × 0.5^(Defence/8)`，最低伤害为 1 |
| **双向碰撞** | 玩家子弹 ↔ 敌人、敌人子弹 ↔ 玩家、玩家 ↔ 敌人三种碰撞独立检测 |
| **边界约束** | 玩家移动限制在窗口内，子弹出界自动回收 |

### 敌人体系

| 敌人 | HP | 攻击 | 速度 | 行为 |
|------|----|------|------|------|
| 大摄魂怪 | 2000 | 0.5 | 0.3 (慢) | 发射子弹，上下浮动，镜像朝向 |
| 摄魂怪 | 800 | 0.1 | 0.3 (慢) | 发射子弹，上下浮动，镜像朝向 |
| 蜘蛛 | 1 | 0.01 | 0.7 (快) | 自旋动画，接触伤害，身体脆弱 |

敌人按照游戏时间分阶段刷新：**大摄魂怪**（0s+）→ **蜘蛛**（20s+）→ **摄魂怪**（35s+），每波随机 1~2 只。

### 道具系统

| 道具 | 来源 | 效果 |
|------|------|------|
| 巧克力蛙 | 击杀任意敌人掉落 | 恢复 4 HP |
| 时间转换器 | 第 40/30/20/10 秒生成 | 命中后所有敌人进入混乱（左右镜像抖动） |
| 福灵剂 | 第 38 秒生成 | 拾取后存为库存，点击按钮使用：恢复 20 魔力 + 30 HP，永久 +1 防御，6 秒金色光晕 |

### 思想泡泡

每 5 秒生成一个随机泡泡，漂浮于场景中。射击击中后触发对应效果：

- **增益泡泡（GainMana）** — 角色跳跃 + 蓝白色荧光闪烁，魔力恢复
- **减益泡泡（LoseMana）** — 角色原地抖动，魔力扣除 10

## 场景与音效

| 场景 | 画面 | 配乐 |
|------|------|------|
| 开始菜单 | 霍格沃茨背景 + 标题 Logo | *Hedwig's Theme* |
| 游戏主场景 | 城堡背景，HP/Mana 双状态条 | 追击主题 OGG |
| 胜利结算 | 胜利场景图 + "Victory" 标题 | 胜利音效 |
| 失败结算 | 失败场景图 + "GAME OVER" | 失败音效 |
| 操作说明 | Info 按钮弹出可滚动图文对话框 | 按钮点击音效 |

所有 UI 按钮（开始 / 介绍 / 退出 / 再来一次 / 福灵剂）均有**悬停、点击音效反馈**。

---

## 操作方式

| 输入 | 动作 |
|------|------|
| `W` `A` `S` `D` | 四向移动，自动切换角色朝向 |
| 鼠标按住拖动 | 向鼠标所在方向连续射击 |

---

## 游戏流程

```
开始菜单 → 点击 START → 50 秒倒计时开始
    ├── HP 归零 → 失败结算 → 再来一次
    └── 撑过 50 秒 → 胜利结算
```

---

## 构建运行

```bash
# 使用 Qt Creator 打开即可编译运行
# 或命令行：
cd src
qmake Expecto_Patronum.pro
make
```

**依赖**：Qt 6 (Core / Gui / Widgets / Multimedia)，C++17

## 项目结构

```
Expecto_Patronum-qt/
├── src/
│   ├── Expecto_Patronum.pro   # Qt 项目文件
│   ├── res.qrc                # 资源索引
│   ├── gamewidget.ui          # UI 布局
│   ├── main.cpp               # 入口
│   ├── gamewidget.*           # 游戏主控（场景管理、碰撞、定时器）
│   ├── gameprogress.*         # 单例进度控制器（敌人生成节奏）
│   ├── player.* / playerbullet.*    # 玩家 & 子弹
│   ├── enemybase.* / enemybullet.*  # 敌人基类 & 敌人子弹
│   ├── dementor.* / bigdementor.* / spider.*  # 三种敌人
│   ├── chocolatefrog.* / timeturner.* / felix_felicis.* / thoughtbubble.*  # 道具
│   ├── gameobject.*           # 游戏对象基类
│   └── mygv.*                 # 自定义 GraphicsView
└── assets/
    ├── *.png / *.jpg          # 角色、场景、UI 图片
    └── *.ogg / *.mp3          # 配乐与音效
```
